Game Of Luck Information

This is a card matching game built in Kotlin for Android devices. Players need to find two matching cards out of three options. The game includes a high score tracking system and provides players with three lives. It features a fun art style and includes a leaderboard using a database so you can track the furtherst a player has gone without losing all of their lives.
